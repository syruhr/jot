# Jot ✨

A frictionless Chrome extension for capturing thoughts, managing todos, and staying focused.

## Features
- ⚡ **Quick Capture** — Open, type, Enter. Done.
- 📝 **Sticky Notes** — Colorful, pinnable, editable cards
- ✅ **Todos** — Satisfying completion animations with confetti
- ⏱️ **Pomodoro Timer** — Focus sessions with circular progress
- 🎯 **Daily Focus** — Set your intention for the day
- 🔢 **Quick Math** — Type `= 2+2` for instant calculation
- 📥 **Export** — Download everything as Markdown

## Setup
```bash
npm install
npm run build
```

Then load `dist/` as an unpacked extension in Chrome (`chrome://extensions` → Developer mode → Load unpacked).

## Development
```bash
npm run dev
```
